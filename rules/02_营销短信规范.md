# 营销短信合规规范

> 本文档适用于所有带有推广、营销、活动宣传等性质的短信模板审核。包括但不限于：产品推广、活动通知、优惠券发放、会员权益推广等。

---

## 一、基本原则

**所有营销短信必须遵循以下核心原则：**

1. **真实性** — 所有产品信息、收益描述、数据引用必须真实可查
2. **完整性** — 不得隐瞒关键信息，不得使用"详见"等方式省略重要条款
3. **适当性** — 不得对老年人、学生等特定脆弱群体进行过度营销
4. **可识别性** — 消费者应能轻易识别该短信为商业推广信息

---

## 二、必含信息要素

每条营销短信**必须**包含以下全部要素：

| 要素 | 说明 | 示例 |
|------|------|------|
| 发送方名称 | 企业全称或注册商标 | 【XX金融】 |
| 产品/活动名称 | 明确告知营销对象 | "新手专享理财" |
| 核心权益/规则 | 简要说明参与条件 | "限新用户，首投满1万返88元" |
| 有效期 | 活动或优惠的时间限制 | "活动截止X月X日" |
| 退订方式 | 告知如何取消订阅 | 回复"T"退订 |
| 客服联系方式 | 有效客服电话或渠道 | 400-XXX-XXXX |

---

## 三、收益承诺类规定（重点）

### 3.1 允许表述

- 可使用历史业绩数据，但需注明"过往业绩不代表未来表现"
- 可使用区间收益描述，需有充分依据
- 可使用"预期收益"、"参考收益"等明确标注性质的词汇

**示例：**
```
【XX金融】季稳盈理财计划，历史年化收益3.5%-4.2%（注：过往业绩不代表未来表现），
限售至3月31日，新用户首投有礼！退订回T，详询400-XXX-XXXX
```

### 3.2 禁止表述（高危）

| 禁止类型 | 违规示例 | 合规替代 |
|---------|---------|---------|
| 虚假高收益承诺 | "稳赚不赔，月收益30%" | "历史年化收益区间3%-5%，投资有风险" |
| 模糊承诺 | "收益率超高"、"稳赚" | 使用具体数值或区间 |
| 隐藏风险 | "零风险高收益" | 明确标注"投资有风险" |
| 绝对化用语 | "保证收益"、"100%安全" | 使用"预期"、"参考"等限定词 |
| 虚假稀缺性 | "仅剩最后3个名额"（实际充足） | 如实描述库存情况 |

### 3.3 收益描述计算公式

如短信中涉及收益计算，必须：
- 使用保守估算参数
- 不得使用极端值最大化了收益
- 如有费用扣除，需同步说明

---

## 四、费用与条款规定

### 4.1 必须明示的费用

- 如涉及手续费、管理费、提现费等任何费用，必须明确告知
- 不得使用"无任何费用"等绝对表述（除非经核实确实免费）

### 4.2 限制条件

- 优惠活动的参与条件必须完整列出
- 不得以"详见条款"、"条件适用"等模糊表述省略关键限制
- 如有最低投资额、锁定期等要求，必须明确说明

---

## 五、外部链接规定

### 5.1 链接报备要求

所有营销短信中的链接（含短链接）**必须**提前报备：

1. 链接指向的页面内容需经法务审核
2. 落地页需与短信承诺内容一致
3. 链接需使用公司域名或经认证的短链服务商

### 5.2 禁止的链接类型

- 未经报备的第三方短链接（如 bit.ly、t.cn 等公网短链）
- 跳转到未经审核的落地页
- 含追踪参数的虚假链接
- 钓鱼风险链接（需通过安全检测）

### 5.3 报备格式

```
链接：[URL]
落地页标题：[标题]
落地页核心内容：[摘要]
审核状态：[已过法务/待审核]
```

---

## 六、用户画像与定向营销

- 向65岁以上老年人发送营销短信，需额外审核子女通知机制
- 向在校学生群体发送金融产品营销短信，原则上禁止（除助学金/勤工俭学类）
- 夜间（21:00—次日8:00）原则上不发送营销短信

---

## 七、禁止行为清单

| 类别 | 禁止行为 | 违规等级 |
|------|---------|---------|
| 收益承诺 | 保证收益、承诺回报、虚假高收益 | 🔴 严重 |
| 内容真实性 | 虚构活动、夸大权益、伪造数据 | 🔴 严重 |
| 链接 | 未经报备链接、跳转未审核页面 | 🔴 严重 |
| 定向人群 | 向学生/老年人过度营销 | 🟡 较重 |
| 时间频次 | 短时间内高频发送 | 🟡 较重 |
| 退订 | 退订后仍继续发送 | 🔴 严重 |
| 伪装 | 伪装成通知/验证码短信发送营销内容 | 🔴 严重 |

---

## 八、审核检查清单

- [ ] 发送方名称是否真实、准确
- [ ] 产品/活动信息是否完整
- [ ] 是否有收益承诺，如有是否合规（不得保证收益、不得虚假夸大）
- [ ] 限制条件是否完整披露
- [ ] 费用信息是否如实告知
- [ ] 链接是否已报备并通过审核
- [ ] 是否包含退订方式和客服电话
- [ ] 是否存在针对老年人/学生的过度营销
- [ ] 发送时间是否合规
- [ ] 短信内容是否与落地页一致

---

## 九、合规等级

| 等级 | 说明 | 处理方式 |
|------|------|---------|
| 🟢 合规 | 完全符合规范要求 | 通过 |
| 🟡 整改 | 收益描述不当/链接未报备/信息不完整 | 退回修改后复审 |
| 🔴 违规 | 保证收益/虚假宣传/未报备链接/伪装发送 | 立即驳回并追责 |
