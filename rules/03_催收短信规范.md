# 催收短信合规规范

> 本文档适用于所有催收类短信模板审核。包括：逾期提醒、欠款催缴、还款引导、律师函通知、征信影响告知等一切与债务催收相关的短信。

> ⚠️ **重要提示**：催收短信合规是金融行业监管重点，本规范参照《银行业金融机构催收指引》、《互联网金融催收自律公约》以及相关法律法规制定。

---

## 一、基本原则

### 1.1 合法合规

- 催收方式不得违反法律法规
- 不得采用恐吓、威胁、骚扰、欺诈等方式催收
- 不得伪造法律文书或冒充公职人员催收

### 1.2 文明理性

- 语言表达须文明、理性、专业
- 不得使用辱骂、歧视性语言
- 不得进行人身攻击或威胁人身安全

### 1.3 保护隐私

- 不得泄露债务人隐私信息（身份证号、住址、具体欠款金额等）
- 严禁向第三方（除担保人/共同借款人外）透露债务信息
- 短信内容不得暴露债务人的债务状况

### 1.4 比例适当

- 催收频次应与欠款严重程度相匹配
- 不得在短时间内高频次发送催收短信
- 逾期初期以提醒为主，严重逾期以实质性催收为主

---

## 二、禁止用语与行为（高危红线）

### 2.1 恐吓威胁类（🔴 严重违规）

以下任何一条均为**一票否决项**，直接驳回：

| 禁止类型 | 违规示例 | 性质 |
|---------|---------|------|
| 威胁人身安全 | "不还钱就弄死你"、"小心你家人安全" | 刑事犯罪 |
| 恐吓上门 | "再不还钱明天就上门"、"已定位你的地址" | 违法催收 |
| 威胁公开信息 | "不还钱就让你社死"、"让你的亲戚朋友都知道" | 侵权违法 |
| 冒充公职身份 | "我是法院执行的"、"公安已立案" | 伪造公文 |
| 伪造法律文书 | "这是法院传票"、"已对你发起诉讼" | 欺诈 |

### 2.2 欺诈类（🔴 严重违规）

| 禁止类型 | 违规示例 |
|---------|---------|
| 伪造逾期费用 | "你已欠费8万"（实际2万） |
| 伪造还款期限 | "今天必须还，否则上报征信"（实际还有3天宽限期） |
| 虚假承诺 | "先还一半就撤诉"（无权撤诉） |
| 冒充熟人 | "我是你朋友XXX介绍的" |

### 2.3 骚扰类（🟡 违规）

| 禁止类型 | 说明 |
|---------|------|
| 重复内容连发 | 短时间内发送多条相同内容催收短信 |
| 无实质内容 | 仅包含"速回电"、"接电话"等无意义内容 |
| 高频发送 | 单日催收短信超过3条（严重逾期除外） |
| 深夜发送 | 夜间（21:00—次日8:00）发送催收短信 |

### 2.4 歧视性语言（🟡 违规）

- 不得使用"老赖"、"穷鬼"等歧视性称谓
- 不得使用"借钱不还还装"、"不要脸"等侮辱性表达
- 不得对债务人的职业、身份进行嘲讽

## 三、合规用语指引

### 3.1 逾期提醒类（温和语气）

适用于逾期初期（1-7天）：

```
【XX金融】尊敬的客户，您的贷款（合同号：XXXX）已逾期X天，剩余应还金额XXX元。
为避免产生更多逾期费用，请尽快还款。如有疑问请致电：400-XXX-XXXX
```

### 3.2 催收提醒类（正式语气）

适用于逾期中期（8-30天）：

```
【XX金融】您的贷款已逾期XX天，请尽快清偿欠款。逾期将影响您的征信记录。
还款金额：XXX元，最低还款：XXX元。客服协助热线：400-XXX-XXXX，退订回T
```

### 3.3 严重逾期告知类（严肃语气）

适用于逾期严重（30天以上）：

```
【XX金融】您在我司的贷款已严重逾期，欠款金额XXX元。长期逾期将严重影响个人征信，
并可能导致法律诉讼。为避免进一步法律后果，请尽快联系客服协商还款方案。
客服热线：400-XXX-XXXX（工作日9:00-18:00）
```

### 3.4 第三方催收机构短信要求

若催收短信由第三方催收机构发送，须满足：

- 短信签名须使用催收机构名称，不得使用原金融机构名称
- 须在首次短信中说明受何机构委托催收
- 须提供原债权方信息

**示例：**
```
【XX催收中心】您有一笔由XX金融机构转让的债务，欠款XXX元，逾期XX天。
受XX金融委托进行欠款催收，请联系：400-XXX-XXXX 协商还款
```

## 四、必含信息要素

| 要素 | 说明 | 备注 |
|------|------|------|
| 发送方名称 | 使用金融机构或催收机构真实名称 | 不得伪装 |
| 欠款金额 | 如实告知（不得伪造） | 可选择告知区间 |
| 逾期天数 | 准确计算 | 不得虚构 |
| 还款方式 | 提供可操作的还款渠道 | 不得只催不帮 |
| 客服联系方式 | 有效联系方式 | 不得设置障碍 |
| 退订方式 | 须提供退订选项 | 不得强制接收 |

---

## 五、禁止附加条件

催收短信中**不得**出现以下附加条件：

- 要求债务人承认债务（"回复Y表示承认欠款"）
- 要求债务人放弃申诉权利
- 要求债务人提供与还款无关的个人信息
- 捆绑销售其他产品

---

## 六、特殊群体保护

### 6.1 老年人保护

- 面向65岁以上老年人的催收短信，语气须更加温和
- 须主动提示可由子女/家属协助处理
- 严禁针对老年人使用任何恐吓性语言

### 6.2 困难群体

- 债务人明确告知经济困难时，应引导至协商还款渠道
- 不得以"贫困证明无效"等方式拒绝合理协商

---

## 七、审核检查清单

- [ ] 是否存在恐吓、威胁用语
- [ ] 是否存在欺诈内容（伪造金额/期限/法律后果）
- [ ] 是否冒充公职身份或伪造法律文书
- [ ] 语言是否文明理性，无歧视性用语
- [ ] 欠款信息是否如实描述
- [ ] 是否提供有效还款渠道和客服电话
- [ ] 发送时间是否合规（非夜间高频）
- [ ] 是否保护债务人隐私（未泄露详细信息）
- [ ] 第三方催收是否使用正确署名
- [ ] 是否包含退订选项

---

## 八、合规等级

| 等级 | 说明 | 处理方式 |
|------|------|---------|
| 🟢 合规 | 完全符合规范要求 | 通过 |
| 🟡 整改 | 语气不当/频次过高/信息不完整 | 退回修改 |
| 🔴 违规 | 恐吓威胁/欺诈/冒充公职 | 立即驳回并启动调查 |

---

## 九、违规处理机制

发现违规催收短信处理将导致：

1. 立即停止该批次短信发送
2. 追溯已发送短信范围，评估影响
3. 通报批评并追究相关人员责任
4. 情节严重者终止合作并追究法律责任
5. 纳入合作催收机构年度考评一票否决项
